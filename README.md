# SetupNotebook

A collection of setup notes and tools for my 2005 Lotus Elise that is run as a NASA TT3 car.

## Tools

- **Chassis** - Edit and save chassis, corners, tires, and related setup
- **Drivetrain** - Dyno chart, NASA Avg HP, optimal shift points, transmission and speed chart
- **Brake Bias** - Calculate brake force distribution
- **Toe vs Suspension Travel** - Visualize bump steer through suspension travel
- **MK60E5 Z4 GT3 CAN Generator** - Generate CAN messages for ABS/TCS configuration
- **Lotus ABS Code Decoder** - Decode EBC430 diagnostic trouble codes

## Usage

Open `index.html` in a browser. No build step required.
